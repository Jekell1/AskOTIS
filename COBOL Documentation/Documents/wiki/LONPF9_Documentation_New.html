Overview

LONPF9 specializes in processing insurance-related transactions, including insurance premium collections and insurance disbursements.

Key Functions

  • Insurance payment processing
  • Premium collection handling
  • Insurance claim disbursements
  • Insurance balance updates

Integration with APIPAY

APIPAY calls LONPF9 when the transaction code is "IN" or "ID":

IF (HOLD-BP-TRCD = "IN" OR "ID")
   MOVE "LP/LONPF9" TO FORM-NAM

Transaction Types Handled

Code Description Purpose
IN Insurance payments Premium collections and insurance payments
ID Insurance disbursements Claims payments and insurance refunds
