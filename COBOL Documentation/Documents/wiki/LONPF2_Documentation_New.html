Overview

LONPF2 processes wire transfers and other electronic payment transactions, handling high-value and electronic fund transfers.

Key Functions

  • Wire transfer processing
  • Electronic payment handling
  • ACH and EFT processing
  • High-value transaction validation

Integration with APIPAY

APIPAY calls LONPF2 when the transaction code is "WI" or "EL":

IF (HOLD-BP-TRCD = "WI" OR "EL")
   MOVE "LP/LONPF2" TO FORM-NAM

Transaction Types Handled

Code Description Purpose
WI Wire transfers High-value electronic fund transfers
EL Electronic payments ACH, EFT, and other electronic transactions
