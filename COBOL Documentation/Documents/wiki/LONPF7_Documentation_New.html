Overview

LONPF7 handles repossession-related transactions and other miscellaneous transaction types that don't fit into standard payment processing categories.

Key Functions

  • Repossession transaction handling
  • Other/miscellaneous transaction processing
  • Asset recovery processing
  • Legal status updates

Integration with APIPAY

APIPAY calls LONPF7 when the transaction code is "RE", "O2", or "OT":

IF (HOLD-BP-TRCD = "RE" OR "O2")
   MOVE "LP/LONPF7" TO FORM-NAM

Transaction Types Handled

Code Description Purpose
RE Repossession transactions Asset recovery and repossession processing
O2 Secondary other transactions Alternative miscellaneous processing
OT General other transactions Catch-all for non-standard processing
