Overview

LONPFC handles check-based transactions, including check processing, validation, and disbursement operations.

Key Functions

  • Check transaction processing
  • Check validation and verification
  • Check disbursement handling
  • Bank reconciliation support

Integration with APIPAY

APIPAY calls LONPFC when the transaction code is "CK" or "CD":

IF (HOLD-BP-TRCD = "CK" OR "CD")
   MOVE "LP/LONPFC" TO FORM-NAM

Transaction Types Handled

Code Description Purpose
CK Check payments Processing of check payments and deposits
CD Check disbursements Check writing and disbursement processing
