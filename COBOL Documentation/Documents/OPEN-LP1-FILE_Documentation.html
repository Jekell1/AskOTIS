Program Overview

OPEN-LP1-FILE opens the loan payment history file, which is used for recording and accessing payment transactions for loans in the APIPAY batch process. This file is required for reading and updating payment history data.

Transaction Types Supported

  • Not transaction-based; supports all operations requiring loan payment file access.

Input Parameters

  • None directly; uses system configuration for file path.

Output Fields

  • None directly; makes the loan payment file available for subsequent operations.
