C-DATE-COMPARE - Date Comparison Utility

Location: .\APIPAY_Inlined.CBL
Generated on: July 22, 2025
Program Type: Date/Time Processing Utility
Section: C-DATE-COMPARE SECTION

Program Overview

C-DATE-COMPARE is a utility that compares two dates (SYS-DATE and NUM-DATE) and returns a result indicating their relationship. This utility has been simplified in recent versions to use direct numeric comparison instead of Julian date conversion.

Input Parameters

  • SYS-DATE (PIC 9(8)): First date for comparison in YYYYMMDD format
  • NUM-DATE (PIC 9(8)): Second date for comparison in YYYYMMDD format

Output Fields

  • ELAPSED-RESULTS (PIC X): Comparison result
  • "E": SYS-DATE equals NUM-DATE
  • "G": SYS-DATE is greater than NUM-DATE
  • "L": SYS-DATE is less than NUM-DATE

Processing Logic

  1. Direct comparison of SYS-DATE with NUM-DATE
  2. Set ELAPSED-RESULTS based on comparison
  3. Simplified logic implemented 2017/02/02

Technical Implementation

IF SYS-DATE = NUM-DATE
   MOVE "E" TO ELAPSED-RESULTS
ELSE
IF SYS-DATE > NUM-DATE
   MOVE "G" TO ELAPSED-RESULTS
ELSE
   MOVE "L" TO ELAPSED-RESULTS

  • MJD 151110: Fixed for CCYY format
