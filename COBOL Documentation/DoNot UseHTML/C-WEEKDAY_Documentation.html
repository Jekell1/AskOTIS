Program Overview

C-WEEKDAY is a utility that determines the day of the week for any given date. It converts the input date to Julian format and uses modular arithmetic to calculate the day of the week, returning a numeric value from 1 (Sunday) to 7 (Saturday).

Input Parameters

  • NUM-DATE (PIC 9(8)): Input date in YYYYMMDD format

Output Fields

  • DAY--OF-WEEK (PIC 9): Day of week (1=Sunday, 2=Monday, ..., 7=Saturday)

Processing Logic

  1. Convert NUM-DATE to Julian date
  2. Add 1 to Julian date for proper day alignment
  3. Divide by 7 and use remainder to determine day of week
  4. Add 1 to remainder to get 1-7 range

Technical Implementation

MOVE NUM-DATE TO P-DATE-IN
PERFORM C-CALL-JUL
MOVE P-JULIAN-DATE TO WS-JUL-WK
COMPUTE WS-JUL = WS-JUL-WK + 1
DIVIDE WS-JUL BY 7 GIVING WS-DIV-RESULT REMAINDER WS-MOD-7
COMPUTE DAY--OF-WEEK = WS-MOD-7 + 1
