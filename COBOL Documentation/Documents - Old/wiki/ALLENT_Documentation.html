Program Overview

ALLENT is a comprehensive window-based program that handles General Ledger allocation entry for financial transactions. It provides an interactive interface for users to distribute monetary amounts across multiple GL accounts with proper validation and audit controls.

Key Responsibilities

  • GL Allocation Management: Allows entry and modification of GL account allocations
  • Amount Distribution: Distributes transaction amounts across up to 5 GL accounts
  • Balance Verification: Ensures allocations balance to the disbursement amount
  • Account Validation: Validates GL account numbers and branch information
  • User Interface: Provides intuitive window-based data entry

Allocation Table

The program manages an allocation table with up to 5 entries, each containing:

  • Line Number: Sequential line identifier (1-5)
  • GL ID: General Ledger identifier (12 characters)
  • Branch: 4-digit branch number
  • Account: 7-digit account number
  • Description: Account description (36 characters)
  • Amount: Allocation amount with 2 decimal places

Validation Rules

Important Validation Rules:

  • Zero Amount Check: No zero amounts allowed (PR# 2662 CENTRIX)
  • Balance Requirement: Total allocations must equal disbursement amount
  • Account Existence: All GL accounts must exist and be valid
  • Branch Authorization: Checks branch permissions for GL entries
  • Audit Integrity: Validates table integrity before exit

Audit Controls

The program includes comprehensive audit controls:

  • Entry Audit: Verifies non-zero amounts have corresponding GL accounts
  • Balance Audit: Ensures allocations don't exceed disbursement amount
  • Table Validation: Checks for incomplete or invalid entries
  • Error Prevention: Prevents out-of-balance GL entries
