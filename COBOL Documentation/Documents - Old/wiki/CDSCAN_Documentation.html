Program Overview

CDSCAN is a comprehensive window-based scanning program that provides lookup and selection capabilities for various code files throughout the system. It supports numerous code types and provides an intuitive interface for code selection and validation.

Key Responsibilities

  • Code Scanning: Provides lookup capabilities for various system codes
  • Window Interface: Interactive window-based code selection
  • Multi-Type Support: Handles numerous different code types
  • Field Selection: Advanced field search and selection capabilities
  • Branch Filtering: Branch-specific code filtering when appropriate

Purpose and Functionality

CDSCAN serves as the primary code lookup utility, providing:

  • Interactive scanning of code files
  • Selection and validation of system codes
  • Branch-specific code filtering
  • Field-based search capabilities
  • Window-based user interface
  • Support for over 25 different code types
