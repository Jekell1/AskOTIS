Program Overview

PAYOFX is a critical calculation utility that computes prior loan payoff amounts without updating any loan records. This program is called by numerous loan processing programs to determine accurate payoff figures for various scenarios including renewals, refinances, and standard payoffs.

Key Responsibilities

  • Payoff Calculation: Computes accurate payoff amounts based on specified dates
  • Interest Calculation: Calculates accrued interest up to payoff date
  • Rebate Processing: Handles various types of rebates (insurance, service charges, etc.)
  • Fee Assessment: Includes applicable fees and charges in payoff amount
  • Multi-Program Integration: Provides payoff services to multiple calling programs
  • Non-Destructive Operation: Performs calculations without updating loan records

Important: PAYOFX is a calculation-only program that does not update any loan records. It provides payoff information that calling programs can use for decision-making and processing.

Calculation Logic

Payoff Calculation Process

  1. Loan Validation
    • Verify loan account exists and is valid
    • Check loan status and payoff eligibility
    • Validate payoff date parameters
  2. Interest Calculation
    • Calculate accrued interest to payoff date
    • Apply appropriate interest rate and formula
    • Handle special interest calculation rules
  3. Fee Assessment
    • Include outstanding late charges
    • Add applicable collection fees
    • Calculate any special charges
  4. Rebate Calculation
    • Calculate insurance rebates
    • Determine service charge rebates
    • Apply finance charge rebates
    • Handle special rebate formulas (including SP-SCFRMLA = 54)
  5. Net Amount Determination
    • Sum all amounts due
    • Subtract applicable rebates
    • Calculate final net payoff amount

Special Calculation Features

  • Rescission Handling: Special processing for rescission scenarios (LN-POCD)
  • MIP Processing: Monthly Insurance Premium calculations
  • O3 Rebate Logic: Special rebate calculation for O3 type
  • Formula 54 Support: Enhanced service charge formula with 4% addition
  • Auto Renewal Validation: Maker comparison for automatic renewals
