Program Overview

MAKERS is a comprehensive window display program that provides detailed information about loan makers (borrowers, co-signers, and endorsers). It displays borrower personal information, employment details, collateral information, and contact phone numbers in an organized, user-friendly interface accessible through Control-N navigation.

Key Responsibilities

  • Borrower Information Display: Shows comprehensive details for up to 4 loan makers
  • Employment Data: Displays current employment status, position, and income information
  • Contact Information: Shows home and work phone numbers with extensions
  • Collateral Display: Presents detailed collateral information including vehicles and household goods
  • Phone Navigation: Provides quick access to phone number management systems
  • Data Integration: Consolidates information from multiple data sources
  • Window Management: Provides floating window interface with proper cleanup

Phone Program Selection Logic

  • Program Path: "WI/PHONE2" or "WI/PHONES" set in VDU-CTRL-N-PROG
