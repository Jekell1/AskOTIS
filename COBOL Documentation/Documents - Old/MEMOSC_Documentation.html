Program Overview

MEMOSC is a comprehensive window-based memo management program that provides inquiry and input capabilities for various types of memos in the loan processing system. It supports multiple memo types including loan memos, borrower memos, dealer memos, collateral memos, and reversal memos with comprehensive password protection and audit controls.

Key Responsibilities

  • Memo Entry and Editing: Provides user interface for entering and modifying memo text
  • Memo Type Management: Handles different memo types with appropriate validation
  • Password Protection: Implements security controls for memo modification and deletion
  • Audit Trail: Maintains complete audit trail of memo changes
  • Date Validation: Enforces business rules for memo dating and backdating
  • Reversal Memo Support: Special handling for transaction reversal memos
  • Collection Integration: Specialized collection memo processing
